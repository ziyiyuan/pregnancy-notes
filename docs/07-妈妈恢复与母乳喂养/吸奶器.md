# 吸奶器

## 这一页解决什么问题

何时需要吸奶器、如何选购使用和避免损伤。

## 结论先行

- **不是待产必买**；母乳顺利时，住院期可用手挤或医院借泵。
- 适用：母婴暂时分离、返岗背奶、涨奶需辅助排空、乳头破损需调整含乳间隔等。
- 原则：**模拟婴儿节律**，不过度追求「抽干」。

## 类型选择

| 类型 | 适用 |
|------|------|
| 手动 | 偶尔使用、外出应急、价格低 |
| 单头电动 | 单侧、使用频率中等 |
| 双头电动 | 返岗、双侧排空、效率需求高 |

护罩尺寸应贴合乳晕，过大过小均影响效果和舒适度。

## 使用要点

1. 哺乳或温敷后泵奶，泌乳反射更好。
2. 从中低档位开始，疼痛则降档或停。
3. 单次约 **15–20 分钟** 为宜（个体调整），不必延长到不适。
4. 配件 **每次清洗并消毒**（沸水 / 蒸汽 / 消毒锅，按说明）。
5. 储奶容器标注日期时间（见 [储奶](储奶.md)）。

## 什么是正常

- 初用泵出量可能少于直接哺乳，随熟练度改善。
- 两侧产量可不同。

## 什么时候需要咨询

- 泵奶剧痛、乳头水肿呈「喇叭口」。
- 泵不出且乳房越来越胀 → 排查含乳、堵奶、乳腺炎。

## 家庭操作清单

- [ ] 护罩尺寸合适。
- [ ] 清洗消毒流程已建立。
- [ ] 背奶冰包、储奶瓶已备（返岗用）。
