# 乳腺炎

## 这一页解决什么问题

识别乳腺炎并及时处理，避免发展为脓肿。

## 结论先行

- 乳腺炎常见表现：**单侧乳房红、肿、热、痛 + 发热、全身酸痛**。
- **不要停止患侧哺乳**（除非医生明确禁忌）；有效排空很重要。
- 出现高热、症状 12–24 h 不缓解 → **尽快就医**，可能需要抗生素。

## 与堵奶的区别

乳腺炎是淤积进展，常伴感染；全身症状更明显。

## 家长具体怎么做

### 就医前（同时联系医生）

1. **继续哺乳或挤奶**，优先排空患侧。
2. 充分休息、补水。
3. 可温敷促进泌乳反射，哺乳后冷敷减痛。
4. **不要** 自行服用抗生素或退烧药隐瞒症状（就医后遵医嘱）。

### 就医时

- 说明发病时间、体温、乳房哪一侧、是否在用吸奶器。
- 按医嘱完成抗生素疗程，即使症状好转也不要擅自停药。

## 什么时候需要急诊

- 高热寒战、血压下降、意识模糊。
- 乳房皮肤发紫、波动感肿块（脓肿可能）。
- 症状迅速加重。

## 预防

- 纠正含乳、按需排空、避免长时间压迫乳房。
- 涨奶、堵奶及时处理。
