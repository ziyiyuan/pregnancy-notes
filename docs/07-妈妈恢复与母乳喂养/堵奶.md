# 堵奶

## 这一页解决什么问题

乳房局部硬块、排奶不畅时的家庭处理与就医指征。

## 结论先行

- 堵奶常表现为 **局部硬块、压痛**，哺乳后可能部分缓解。
- 处理核心：**有效排空 + 解除淤积**，不是暴力通乳。
- 硬块伴红热、发热 → 按 [乳腺炎](乳腺炎.md) 处理并就医。

## 常见原因

- 含乳浅、吸吮无效、长时间不排空、压迫乳房（过紧文胸、趴睡压胸）、突然减少哺乳等。

## 家长具体怎么做

1. **先哺乳** 患侧，宝宝下巴对准硬块方向有时更有效。
2. 哺乳后仍硬：温敷 → 轻柔按摩 **乳房周围向乳头方向** → 手挤或短时吸奶。
3. 冷敷减轻肿胀。
4. **24 h 内** 重复上述，记录硬块变化。
5. 避免让非专业人员进行强力「通乳」导致水肿、损伤。

## 什么是正常

- 小块淤积在 1–2 次有效排空后变软缩小。

## 什么时候需要就医

- 硬块 **>24–48 h** 不消或增大。
- 皮肤发红、发热、条索状硬结。
- 体温 ≥38℃、全身不适。
- 乳头白点（奶泡）反复、无法排出 — 可请哺乳顾问处理。

## 家庭操作清单

- [ ] 记录硬块位置、出现时间。
- [ ] 哺乳前后乳房软硬对比。
- [ ] 不自行暴力揉捏。

## 本文依据

- 乳汁淤积与乳腺炎鉴别处理常规。
