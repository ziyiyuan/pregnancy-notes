# 盆底恢复

## 这一页解决什么问题

介绍产后盆底肌、漏尿等问题的基础认知和何时需要专业康复。

## 结论先行

- 怀孕和分娩可导致盆底肌松弛；**漏尿、坠胀感** 较常见，但 **不应忽视**。
- **凯格尔运动** 可在医生 / 康复师指导下逐步开始。
- 产后 **6 周复查** 时讨论盆底评估。
- 严重脱垂、持续漏尿 → 寻求 **盆底康复门诊或泌尿妇科**。

## 常见症状

- 咳嗽、大笑、抱娃时 **漏尿**（压力性尿失禁）。
- 阴道坠胀、异物感。
- 性生活不适（恢复后仍持续应咨询）。

## 家长具体怎么做

### 产后早期（0–6 周）

- 以休息为主，**不要** 自行剧烈收腹、绑腹带替代康复。
- 避免便秘（用力排便会加重盆底负担）。
- 可在无不适前提下，按医嘱开始 **轻柔的凯格尔**（收缩肛门和阴道周围肌肉，保持数秒后放松，重复）。

### 6 周后

- 产科复查时主动说明漏尿、坠胀。
- 必要时转诊盆底康复（电刺激、生物反馈、手法等）。
- 逐步恢复低强度运动（散步、游泳等），避免过早高强度跑跳。

## 什么是正常

- 短期轻度漏尿可能随恢复改善。
- 腹部松弛、腹直肌分离需时间；与盆底问题可并存但处理不同。

## 什么时候需要就医

- 漏尿影响生活、不改善。
- 阴道肿物脱出、持续坠痛。
- 发热伴盆腔痛（排除感染）。
