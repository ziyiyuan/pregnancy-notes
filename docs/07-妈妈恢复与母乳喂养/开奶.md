# 开奶

## 这一页解决什么问题

帮助妈妈在产后尽早、有效地建立母乳喂养。

## 结论先行

- ✅ **产后尽早肌肤接触**；**黄金 1 小时**内尝试第一次哺乳。
- ✅ 早期重点是 **含乳姿势 + 频繁吸吮**，不是先算奶量。
- 剖宫产 / 全麻：婴儿可能嗜睡，需在清醒窗内更主动尝试。
- 持续含乳疼痛、宝宝体重 / 尿量异常 → 找 **哺乳顾问或医生**。

## 为什么重要

- 早期吸吮刺激泌乳；哺乳促进子宫收缩、减少产后出血。
- 延迟超过 1 小时再哺乳，婴儿可能更困、更难含住乳头。

## 家长具体怎么做

### 产房内（0–2 小时）

1. 母婴肌肤接触（政策允许时）。
2. 观察宝宝 **清醒信号**（睁眼、寻乳、吸吮手指）。
3. 尝试含乳：腹部贴腹、鼻子对乳头、含住大部分乳晕。
4. 记录是否听到吞咽。

### 住院期间

- **按需哺乳**，通常 8–12 次 / 24 h。
- 每次含乳困难 → 请护士或哺乳顾问 **看姿势**。
- 避免过早添加配方奶（除非医学指征）。
- 记录湿尿布（第 4 天起目标 ≥6/日）。

### 含乳自检

| 项目 | 正确表现 |
|------|----------|
| 含乳 | 乳晕大部分入口，嘴唇外翻 |
| 疼痛 | 初始几秒可忍，不应持续剧痛 |
| 吞咽 | 可听到或看到吞咽 |
| 宝宝 | 面颊鼓起、下巴贴乳房 |

## 什么是正常

- 初乳量少、颜色黄稠。
- 产后 2–5 天乳房胀感增强（「下奶」）。
- 频繁要吃、夜醒吃奶。

## 什么时候需要就医 / 咨询

- 乳头大面积破损、出血。
- 宝宝尿少、体重下降 >10%、嗜睡拒奶。
- 乳房红肿热痛伴发热（见 [乳腺炎](乳腺炎.md)）。
