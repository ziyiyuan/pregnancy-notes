# 恶露

## 这一页解决什么问题

帮助妈妈识别产后子宫排出的恶露是否正常，以及何时需要就医。

## 结论先行

- 恶露是子宫蜕膜脱落排出，**持续约 4–6 周**，量应 **逐渐减少**。
- 颜色通常：**红 → 淡红 / 粉红 → 淡黄 / 白**。
- 突然大量出血、恶臭、发热 → **立即就医**。

## 恶露分期（参考）

| 阶段 | 时间（约） | 颜色 / 性状 |
|------|------------|-------------|
| 血性 | 产后 1–3 天 | 鲜红，可有小血块 |
| 浆液性 | 4–10 天 | 淡红、粉红、褐色 |
| 白色 | 10 天–6 周 | 淡黄或白色，量渐少 |

个体差异大，不必机械对号入座，重点看 **趋势**。

## 家长具体怎么做

- 使用 **产褥垫 / 卫生巾**，勤更换。
- 每日留意：量、颜色、气味、是否血块增多。
- 哺乳时宫缩可能暂时增加排出，属常见。
- 不用阴道冲洗、不用棉条（国内常规也不建议）。

## 什么是正常

- 量逐日减少。
- 无恶臭（轻微血腥味可接受）。
- 无持续高热。

## 什么时候需要就医

- **1 小时内** 浸透整片卫生巾，或反复大血块。
- 恶露 **停止后又大量鲜红出血**（警惕晚期产后出血）。
- 发热 ≥38℃、腹痛、恶露 **恶臭**。
- 头晕、心慌、面色苍白（失血过多）。

## 家庭操作清单

- [ ] 备足卫生巾 / 产褥垫。
- [ ] 知道急诊电话和分娩医院产科电话。
- [ ] 记录每日出血量变化（异常时就医用）。
