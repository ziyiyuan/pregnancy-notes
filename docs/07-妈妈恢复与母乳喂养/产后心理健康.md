# 产后心理健康

## 这一页解决什么问题

帮助家庭识别产后情绪波动是否正常，以及何时需要专业帮助。

## 结论先行

- 产后 **情绪波动很常见**（「baby blues」），多在 **2 周内** 缓解。
- 若 **持续低落、焦虑、失眠、自责、伤害自己或宝宝念头** → **尽快寻求专业帮助**。
- 爸爸 / 家人同样可能焦虑，应互相观察和支持。

## 常见类型

| 类型 | 特点 | 处理 |
|------|------|------|
| Baby Blues | 产后数天哭、易怒、疲惫，2 周内好转 | 休息、分担、倾诉 |
| 产后抑郁 | 持续 >2 周，兴趣丧失、绝望、严重失眠 | **精神科 / 心理科** |
| 产后焦虑 | 过度担心宝宝、恐慌、躯体紧张 | 专业评估 |
| 产后精神病 | 罕见，幻觉、妄想，紧急 | **立即急诊** |

## 家长具体怎么做

### 妈妈

- 争取 **连续睡眠**（家人轮班）。
- 减少完美主义；接受「够好」的照护。
- 保持基本社交联系，不必封闭在家。
- 适度户外活动、日照（身体状况允许时）。

### 家人

- 主动承担家务和夜间事务。
- 不说「你想开点」「别人都能带」。
- 观察：是否不愿抱娃、是否反复表达无价值感。
- 鼓励就医，陪同就诊。

## 危险信号（立即求助）

- 想伤害自己或宝宝。
- 听见不存在的声音、坚信被害。
- 完全无法进食、睡眠、照顾自己和宝宝。

拨打心理危机热线或急诊；国内可 120 / 当地精神卫生中心。

## 哺乳与用药

- 产后抑郁治疗 **不应因哺乳而拖延**；多数药物有哺乳兼容选项，由医生权衡。

## 家庭操作清单

- [ ] 家人知道心理危机热线。
- [ ] 分工模板已落实，保证妈妈睡眠。
- [ ] 情绪低落 >2 周已预约专业评估。
