# 回奶

## 这一页解决什么问题

决定停止哺乳时，如何相对舒适、安全地减少泌乳。

## 结论先行

- 回奶宜 **循序渐进**（除非医学需立即停奶）。
- 核心：**逐渐减少排空频率**，让乳房逐步适应。
- 胀痛明显可冷敷、穿支撑文胸；**不要** 暴力挤空或突然完全停止导致严重涨奶 / 乳腺炎。
- 需用药回奶 → **遵医嘱**（不同药物禁忌不同）。

## 渐进式回奶步骤

1. 先去掉 **最不常用的一顿**（或缩短单侧时间），观察 2–3 天。
2. 逐步拉长哺乳间隔，用挤奶仅排到 **舒适**（不必抽干）。
3. 减少液体摄入无科学依据，正常饮水即可。
4. 乳房仍胀：冷敷、布洛芬类可按说明书或医嘱（哺乳期间用药需咨询）。

## 不建议

- 绑紧乳房、暴力通乳「回奶」。
- 自行服用偏方、不明中药。
- 突然完全停止且不处理胀痛。

## 什么时候需要就医

- 回奶过程中高热、乳房红肿（乳腺炎）。
- 硬块持续不消、剧痛。

## 心理与支持

- 断奶是正常选择，不必内疚；情绪起伏可寻求家人或专业支持。

## 家庭操作清单

- [ ] 已规划渐进时间表。
- [ ] 备冷敷、支撑文胸。
- [ ] 知道乳腺炎警示信号。
