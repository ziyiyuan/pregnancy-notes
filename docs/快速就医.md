# 快速就医

> **不确定时，宁可早联系医生。** 本页为家庭速查，不能替代急诊判断。

## 立即急诊 / 拨打 120

| 症状 | 说明 |
|------|------|
| **发热** | 尤其 **<3 月龄** 腋温 **≥38℃** |
| **呼吸困难** | 呼吸很快、胸凹、呻吟、口唇发紫 |
| **抽搐** | 意识异常、叫不醒 |
| **持续拒奶** | 吃奶明显无力 |
| **严重脱水** | 尿极少、囟门凹陷、精神差、哭无泪 |
| **喷射性呕吐** | 反复、量大 |
| **血便 / 血尿** | 大量或持续 |
| **意外伤害** | 严重跌落、烫伤、窒息、误吞危险品 |

**同时可做**：记录体温、最近 24 h 吃奶与尿布次数；拍照异常部位；带好母子手册和出院小结。

---

## 24 小时内联系医生

- 黄疸 **明显加重**（眼白、全身发黄）
- 湿尿布 **持续减少**
- 体重增长 **不理想** 或下降过多
- 脐部 **红肿、流脓、恶臭**
- 持续哭闹 **无法安抚** 且无明确原因
- **腹泻** 明显、精神差
- 皮疹大片 **渗液、发热**

---

## 可先观察并记录

- 偶尔吐奶，精神、吃奶、尿布正常
- 打嗝、喷嚏
- 轻微脱皮
- 短暂粉色尿渍 + 湿尿布充足
- **惊跳反射**（睡眠中突然动一下）
