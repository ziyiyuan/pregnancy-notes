# 喂养 FAQ

## 母乳够不够？

看 **含乳、吞咽、湿尿布（第4天起≥6/日）、体重、精神**，不是看乳房软不软。详见 [02-吃](../02-新生儿期-0-4周/吃.md)、[0-4周 FAQ](../02-新生儿期-0-4周/FAQ.md)。

## 总想吃奶是不是没奶？

不一定，可能是 **密集喂养期**。无吞咽 + 尿少 → 评估喂养。

## 要不要加水？

通常 **不需要**。见 [02-喝](../02-新生儿期-0-4周/喝.md)。

## 母乳稀没营养？

误区 → [母乳稀就是没营养](../常见误区/母乳稀就是没营养.md)

## 维生素 D？

纯母乳儿通常 **2 周内** 起 400 IU/日（遵医嘱）。见 [02-FAQ](../02-新生儿期-0-4周/FAQ.md) Q10。

## 何时就医？

拒奶、尿少、体重下降过多 → [快速就医](../快速就医.md)
