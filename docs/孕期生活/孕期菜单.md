# 孕期菜单

日常饮食安排，便于备餐与轮换。

---

## 早餐（七选一，可轮换）

| 序号 | 搭配 |
|------|------|
| 1 | 西红柿鸡蛋面 + 煮鸡蛋 1 个 |
| 2 | 卷饼（饼 + 胡萝卜丝 + 蔬菜 + 炒鸡蛋碎）+ 豆浆或小米粥 |
| 3 | 卷饼（饼 + 土豆丝 + 蔬菜 + 炒鸡蛋碎）+ 豆浆或小米粥 |
| 4 | 馄饨 + 开水烫熟小油菜 + 煮鸡蛋 1 个 |
| 5 | 蒸包子 + 蒸粗粮 + 小米粥 + 煮鸡蛋 1 个 |
| 6 | 蒸米糕 + 蒸粗粮 + 西兰花 + 豆浆或小米粥 + 煮鸡蛋 1 个 |
| 7 | 馄饨 + 开水烫熟小油菜 + 煮鸡蛋 1 个 |

**卷饼蔬菜**（任选其一或搭配）：菠菜、油麦菜、花菜、包菜、白菜。
**蒸粗粮**（任选其一或搭配）：玉米，红薯，南瓜，芋头。


> **备餐提示**：食材前一天洗好放冰箱，第二天直接炒。土豆丝需提前泡水。

---

## 加餐

- 香蕉 1 根  
- 橙子 / 橘子 / 苹果 / 梨 1 个（任选）  
- 酸奶 135 g  

---

## 午、晚餐（两餐正餐）

- **比例**：约 1/2 蔬菜、1/4 蛋白质、1/4 主食。  
- **种类**：蔬菜、肉、豆类、鱼等，正常搭配即可。

---

| 下午 | 力蜚能 1 颗 |
