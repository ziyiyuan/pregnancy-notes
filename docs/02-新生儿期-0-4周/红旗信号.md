# 红旗信号

## 这一页解决什么问题

把 0–4 周最需要警惕的情况集中列出，方便家长快速判断是否需要就医。

> 原则：新生儿变化快。只要家长觉得宝宝和平时明显不一样，宁可早联系医生。

## 立即急诊或立刻联系医生

- 发热：尤其是 3 个月以内婴儿体温达到或超过 38℃。
- 呼吸困难、胸凹、呻吟样呼吸、口唇发紫。
- 持续拒奶或吃奶明显无力。
- 明显嗜睡、很难叫醒。
- 抽搐或意识异常。
- 反复喷射性呕吐。
- 大便有大量血，或尿液明显带血。
- 尿量明显减少并伴精神差。
- 脐部周围红肿、流脓、有臭味。

## 24 小时内咨询医生

- 黄疸明显加重，或眼白、躯干、四肢都明显发黄。
- 体重增长不理想或体重下降过多。
- 湿尿布数量持续偏少。
- 持续哭闹难以安抚，但没有明确原因。
- 大便持续水样、次数明显增多。
- 皮肤大片红疹、渗液或破溃。

## 可以先观察但要记录

- 偶尔吐奶，但精神、吃奶、尿布和体重正常。
- 短暂粉色尿酸盐痕迹，但湿尿布数量足够。
- 偶尔打嗝、喷嚏。
- 轻微脱皮。

## 就医前建议记录

- 体温和测量方式。
- 最近 24 小时吃奶次数和奶量 / 含乳情况。
- 湿尿布和大便次数。
- 呕吐次数和样子。
- 精神状态变化。
- 拍照记录皮疹、尿布、大便或脐部异常。

## 本文依据

- 《美国儿科学会育儿百科》关于新生儿危险信号、发热、脱水、黄疸和排泄异常的建议。
- 儿科常规就医警示原则。
