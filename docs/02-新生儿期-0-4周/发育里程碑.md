# 发育里程碑

## 这一页解决什么问题

帮助家长了解 0–4 周新生儿常见发育表现，并知道哪些情况需要在儿保或就医时提出。

## 本阶段特点

新生儿期的发育重点不是“学会很多技能”，而是适应宫外生活：吃奶、睡眠、体温调节、排便排尿、对声音和人脸产生反应。

## 大运动

常见表现：

- 四肢活动多为不协调动作。
- 俯卧时可能短暂尝试抬头。
- 头颈控制能力很弱，抱起时必须托住头颈。

家长可以做：

- 清醒时在成人看护下短时间俯卧练习。
- 抱起、放下、拍嗝时始终支撑头颈。

## 精细动作

常见表现：

- 手常握拳。
- 有抓握反射。
- 手部动作还不是有目的的抓取。

## 语言和听觉

常见表现：

- 主要通过哭声表达需求。
- 对熟悉人声、突然声音可能有反应。

## 社交

常见表现：

- 喜欢看近距离人脸。
- 视距大约在照护者抱着喂奶时的距离更容易看清。
- 可能短暂凝视照护者。

## 红旗提示

如果出现以下情况，建议咨询儿科医生或在儿保时提出：

- 吃奶明显无力。
- 四肢活动明显不对称。
- 对强声音完全无反应。
- 持续异常嗜睡或很难唤醒。
- 抱起时身体特别软或特别僵硬。
