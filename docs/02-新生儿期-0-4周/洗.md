# 洗

## 这一页解决什么问题

帮助家长掌握新生儿擦浴、盆浴、脐带护理、皮肤护理和剪指甲的基本方法。

## 结论先行

- 脐带残端脱落前，通常以擦浴为主。
- 1 岁以内不需要每天洗澡；如果尿布区清洁到位，每周约 3 次全身洗澡通常足够。
- 水温以手腕或肘内侧感觉温热为宜，热水最高出水温度建议不超过 49℃。
- 洗澡时必须全程有成人看护，不能离开。

## 家长具体怎么做

### 擦浴准备

提前放在伸手可及处：

- 温水
- 干净小毛巾或棉柔巾
- 浴巾
- 干净尿布
- 干净衣物
- 低刺激婴儿清洁用品

### 擦浴顺序

1. 用浴巾包住宝宝，只露出正在清洁的部位。
2. 先用不含肥皂的湿毛巾擦脸。
3. 再清洁头部、身体和四肢。
4. 注意脖子、腋下、耳后、大腿根等皱褶。
5. 尿布区最后清洁。
6. 洗后轻轻拍干，不要用力擦。

### 脐带护理

- 保持 **干燥**；尿湿或沾便及时清洁，蘸干。
- 按出院指导用 **碘伏或酒精** 消毒（不同医院建议略有差异）。
- 不要强行扯掉残端；通常 **1–2 周** 自然脱落。
- 穿纸尿裤时可 **折下前端**，避免摩擦脐部。
- **不要** 用脐带护贴长期密闭包裹（除非医生要求）。

### 盆浴

脐部完全愈合后可以尝试盆浴。

- 先接好水，再把宝宝抱入水中。
- 水不要太深。
- 一只手始终支撑宝宝头颈。
- 如果宝宝明显抗拒，可以继续擦浴 1–2 周后再尝试。

## 皮肤护理

- 通常不需要每天使用润肤油、爽身粉。
- 皮肤干燥时，可少量涂无香精、低敏润肤霜。
- 如果皮肤越洗越干，先减少洗澡频率。

## 剪指甲

- 新生儿指甲长得快，前几周可能每周需要修剪 1–2 次。
- 可在宝宝睡着时剪。
- 使用婴儿指甲剪或磨甲器。
- 脚趾甲不需要剪得像手指甲那么短。

## 什么时候需要就医

- 脐部周围红肿、流脓、有臭味。
- 皮肤大片红肿、渗液或宝宝明显疼痛。
- 洗澡后出现异常嗜睡、呼吸困难或体温异常。

## 家庭操作清单

- [ ] 脐带未脱落前以擦浴为主。
- [ ] 洗澡前所有物品放在伸手可及处。
- [ ] 热水温度已测试。
- [ ] 洗澡全过程不离开宝宝。
- [ ] 洗后及时拍干并穿衣保暖。
