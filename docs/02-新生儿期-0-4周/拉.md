# 拉

## 这一页解决什么问题

帮助家长识别新生儿大便从胎便到「正常便」的变化，以及何时需要就医。

---

## 结论先行

- 出生后数日内排 **黑绿黏稠胎便** 是正常的。
- 过渡后，**母乳便** 常呈黄色、稀软、有颗粒，**一天多次** 可正常。
- **配方奶便** 略稠，但仍应软于花生酱。
- 血便、水样腹泻伴精神差、长时间不排胎便 → **就医**。

---

## 大便变化时间线

| 阶段 | 性状 | 时间 |
|------|------|------|
| 胎便 | 黑绿、黏稠、无味 | 出生后 24–48 h 内应开始排 |
| 过渡便 | 绿黄、黄绿 | 胎便减少后 |
| 母乳便 | 黄色、稀软、小颗粒 | Establish 母乳后 |
| 配方奶便 | 黄或褐黄，较稠但软 | 配方奶喂养 |

---

## 什么是正常

- 母乳喂养：大便 **非常松软**，甚至像米糊，**每次奶后都可能排**。
- 配方奶：比母乳便成形，但 **不应干硬**。
- 只要宝宝 **精神好、吃奶好、体重正常**，最后排出的大便 **仍然柔软**（软于花生酱），次数多不一定有问题。
- 几天排一次但 **便软、宝宝舒适** — 部分母乳宝宝可能出现，需结合整体状态（有争议时问医生）。

---

## 家长怎么观察

- 记录 **次数 + 性状**（颜色、稀稠、有无血 / 黏液）。
- 换尿布时 **彻底清洁** 臀区，预防红屁股。
- 母乳妈妈一般 **无需** 为「绿便」盲目忌口；持续异常再咨询。

---

## 危险信号

| 信号 | 说明 |
|------|------|
| 出生后 **48 h 未排胎便** | 需排除肠道问题 |
| **大量血** 或黏液 | 立即就医 |
| **水样腹泻** | 次数超过吃奶次数、水分异常高 |
| 腹泻 + **发热**（尤其 <2 月龄） | 立即就医；仍要定时喂养 |
| 大便 **干硬**、排便痛苦 | 可能摄入不足或脱水 |
| 腹胀明显 + 呕吐 + 精神差 | 就医 |

**腹泻的主要风险是脱水** — 见 [喝](喝.md)、[撒](撒.md)。
