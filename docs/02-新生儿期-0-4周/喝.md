# 喝

## 这一页解决什么问题

解答「新生儿要不要单独喂水」以及如何通过尿尿判断水分是否足够。

---

## 结论先行

- ✅ 健康足月儿在 **母乳或配方奶充足** 时，**通常不需要额外喂水**。
- ✅ 观察 **湿尿布数量和尿色** 比估「喝了多少水」更可靠。
- 发热、腹泻、高温环境下是否补水 → **遵医嘱**，不要自行大量喂水。

---

## 为什么不需要额外喝水

母乳约 87% 是水分；按说明冲调的配方奶也已含水。额外喂水会：

- 占用胃容量，减少奶摄入；
- 对很小婴儿存在 **水中毒** 风险（极端情况）。

---

## 家长具体怎么做

- 保证 **有效吃奶**（见 [吃](吃.md)）。
- 记录 **湿尿布**：出生第 4 天起通常 **≥6 次 / 日**。
- 尿色 **淡黄～黄色** 为常见；越深可能提示摄入偏少或尿液浓缩。
- 天气炎热时，通过 **增加哺乳次数** 补水，而非单独喂水（除非医生另有建议）。

---

## 什么是正常

| 情况 | 说明 |
|------|------|
| 尿量个体差异 | 约 1–3 h 一次，或 4–6 次 / 日均可见 |
| 尿色深浅变化 | 与摄入量有关 |
| 偶尔粉色痕迹 | 可能是尿酸盐结晶；湿尿布够 → 观察；**持续** → 就医 |
| 生病 / 发热时尿少 | 可在正常范围内减少，但仍需关注 |

---

## 危险信号

- 长时间无湿尿布
- 尿量持续明显减少 + 精神差、拒奶
- 尿中带 **血**
- 排尿时明显 **痛苦** 哭闹
- 脱水征：口唇干、囟门凹陷、哭无泪、皮肤弹性差

→ 联系医生或急诊。

---

## 家庭操作清单

- [ ] 未自行给新生儿喂水（除非医嘱）。
- [ ] 每日数湿尿布。
- [ ] 尿色持续很深且奶量可疑 → 评估喂养或咨询。
