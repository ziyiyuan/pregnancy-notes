# 穿

## 这一页解决什么问题

帮助家长判断新生儿穿多少、怎么包裹、睡觉穿什么，以及哪些衣物不建议过早使用。

## 结论先行

- 新生儿通常按“比成人多一层”的原则穿衣。
- 判断冷热优先摸颈背，不只看手脚。
- 睡觉时优先使用连体睡衣或睡袋，避免松散厚被。
- 不会走路前不需要穿鞋。
- 袜子和连脚衣不能过紧，避免影响足部活动。

## 家长具体怎么做

### 日常穿衣

常见组合：

1. 贴身衣物。
2. 纸尿裤。
3. 连体衣或睡衣。
4. 根据室温增加包被或睡袋。

### 判断冷热

- 颈背温暖、无汗：通常合适。
- 颈背出汗、脸红、烦躁：可能过热。
- 胸背偏凉、精神差：需要保暖并观察。

手脚偏凉不一定代表宝宝冷，因为新生儿末梢循环还不成熟。

### 包裹

适度包裹可让部分新生儿更有安全感，但要注意：

- 不要包得过紧。
- 臀部和双腿要能自然活动。
- 不要遮住口鼻。
- 一旦宝宝开始尝试翻身，应停止包裹睡眠。

## 不建议

- 新生儿鞋。
- 过紧袜子。
- 过小连脚衣。
- 睡觉时盖厚重松散被子。

## 什么时候需要观察

- 宝宝经常满头大汗。
- 皮肤出现热疹。
- 手脚被袜口勒出明显痕迹。
- 包裹后哭闹明显加重。

## 家庭操作清单

- [ ] 睡觉不盖松散厚被。
- [ ] 袜口、袖口、裤脚不勒。
- [ ] 包裹时髋部和腿部有活动空间。
- [ ] 根据颈背温度调整衣物。
