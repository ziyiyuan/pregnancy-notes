# 陪

## 这一页解决什么问题

帮助家长理解新生儿哭闹、安抚、互动和屏幕暴露问题。

## 结论先行

- 新生儿哭是沟通方式，不是故意“闹人”。
- 第一个月应尽量及时回应哭闹，不建议刻意训练宝宝哭很久。
- 清醒时可以进行短时间面对面互动。
- 清醒且有人看护时可以进行俯卧练习。
- 2 岁以内应尽量避免屏幕暴露，包括背景电视。

## 宝宝为什么会哭

常见原因：

- 饿了
- 困了
- 尿布湿了
- 太热或太冷
- 想被抱
- 肠胀气或不舒服
- 环境刺激太多

## 家长具体怎么做

### 安抚顺序

1. 看尿布。
2. 判断是否该吃奶。
3. 摸颈背判断冷热。
4. 抱起轻拍或轻轻摇动。
5. 调暗灯光，减少刺激。
6. 轻声说话、唱歌或播放柔和声音。
7. 如果困了，开始哄睡。

### 日常互动

- 让宝宝看照顾者的脸。
- 轻声说话或唱歌。
- 短时间看黑白 / 红色高对比卡。
- 清醒时在成人看护下俯卧练习。

## 屏幕

2 岁以内尽量避免让宝宝看电视、手机和平板。即使电视不是给宝宝看的，背景电视也可能打断宝宝正在进行的观察、玩耍和亲子互动。

## 什么时候需要就医

- 哭声尖锐异常。
- 持续无法安抚，并伴随发热、呕吐、腹胀、血便、精神差。
- 家长直觉觉得宝宝“和平时不一样”。

## 家庭操作清单

- [ ] 每次哭闹先排查饿、困、尿布、冷热。
- [ ] 每天有几次短时间面对面互动。
- [ ] 清醒时进行短时间俯卧练习。
- [ ] 尽量关闭背景电视。
- [ ] 访客接触宝宝前洗手，生病者不接触。
