# 睡

## 这一页解决什么问题

帮助家长理解新生儿为什么睡得碎、醒得频繁，以及第一个月如何在保证安全的前提下哄睡。

## 结论先行

- ✅ 1 岁以内婴儿应仰卧睡眠；半岁以内尤其重要。
- ✅ 睡眠环境应独立、平坦、坚实，不放枕头、厚被、床围和毛绒玩具。
- 新生儿没有稳定昼夜节律，频繁夜醒是正常的。
- 出生后第一个月不建议刻意让宝宝哭很久，应该及时回应。
- 清醒 1–2 小时后，很多新生儿已经需要再次入睡。

## 为什么宝宝会这样

新生儿神经系统和昼夜节律尚未成熟，睡眠分散在全天。饥饿、尿布、冷热、惊跳反射、过度刺激都可能让宝宝醒来或哭闹。

## 家长具体怎么做

### 1. 先保证安全睡眠

- 仰卧放睡。
- 睡在婴儿床、床旁床等独立睡眠空间。
- 床垫坚实，床笠贴合。
- 不放枕头、被子、床围、毛绒玩具。
- 担心冷时优先使用合适厚度的睡袋，而不是松散被子。

### 2. 观察困倦信号

常见信号：

- 打哈欠
- 眼神发直
- 扭动、烦躁
- 吸吮动作增加
- 哭闹

哭闹通常已经是比较晚的困倦信号。

### 3. 清醒 1–2 小时后尝试哄睡

第一个月宝宝清醒时间通常较短。如果拖到过度疲劳，反而更难入睡。

可尝试：

1. 调暗灯光。
2. 减少说话和刺激。
3. 轻轻抱起、轻拍、轻声说话或唱歌。
4. 昏昏欲睡但还没完全睡着时，尝试放回床上。

### 4. 白天和夜晚逐渐区分

- 白天保留自然光和正常生活声音。
- 夜间喂奶、换尿布尽量低光、安静、少互动。
- 不需要在第一个月强行建立固定作息。

## 什么是正常

- 白天睡很多、晚上也频繁醒。
- 每次睡眠长短不一。
- 吃完奶睡着。
- 放下就醒。
- 因惊跳反射突然动一下或醒来。

## 什么时候需要观察

- 总是很难入睡，且明显过度疲劳。
- 每次小睡都很短，但吃奶、精神、尿布和体重正常。
- 夜间完全没有较长睡眠片段，但宝宝整体状态正常。

## 什么时候需要就医

- 呼吸困难、口唇发紫。
- 明显嗜睡、叫不醒。
- 发热。
- 持续拒奶。
- 抽搐或异常动作后难以唤醒。

## 家庭操作清单

- [ ] 婴儿床内无枕头、被子、床围、玩具。
- [ ] 每次睡眠都仰卧放睡。
- [ ] 夜间使用小夜灯，避免强光刺激。
- [ ] 清醒时在成人看护下短时间俯卧练习。
- [ ] 记录明显异常的睡眠、吃奶和精神状态。
