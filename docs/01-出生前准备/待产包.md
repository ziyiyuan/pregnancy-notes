# 待产包

## 这一页解决什么问题

列出入院分娩需要携带的物品。可与医院清单合并使用，**以分娩医院要求为准**。

---

## 证件与资料（三者常备）

| 物品 | 说明 |
|------|------|
| 身份证 | 父母双方 |
| 医保卡 / 社保卡 | 结算用 |
| 《母子健康手册》 | 产检记录 |
| 银行卡 | 押金、自费项目 |
| 手机 + 充电器 + 充电宝 | 联系与支付 |

---

## 妈妈 · 进产房

| 物品 | 数量建议 | 备注 |
|------|---------:|------|
| 产褥垫 / 护理垫 | 若干 | 部分医院提供，可备 |
| 计量卫生巾 / 夜用卫生巾 | 1–2 包 | 产后恶露 |
| 吸管杯 | 1 | 躺着喝水 |
| 能量零食 | 少量 | 巧克力、能量棒（按医院规定） |
| 发圈 | 1 | 长头发必备 |
| 纸巾、湿巾 | 若干 | |

## 宝宝

| 物品 | 数量建议 | 备注 |
|------|---------:|------|
| NB 码纸尿裤 | 1 小包 | 医院可能提供 |
| 湿巾 + 棉柔巾 | 各 1 包 | 棉柔巾洗屁屁更温和 |
| 52 码连体衣 | 2–3 | 出院穿 |
| 包被 / 襁褓 | 1–2 | 视季节 |
| 婴儿帽、袜子 | 各 1 | 视季节和室温 |
| **儿童安全座椅** | 1 | **出院必须**，提前装车 |

| 物品 | 备注 |
|------|------|
| 身份证 | 陪护登记 |
| 换洗衣物、拖鞋 | |
| 薄被或毯子 | 陪护床 |
| 洗漱用品 | |
| 简单食物 | 医院食堂不便时 |

---

## 不建议带进产房的

- 厚重毛毯（易过热）
- 香水、强气味用品
- 过多现金和贵重物品

---

## 打包建议

```
┌─ 证件包（随时可拿）
├─ 妈妈住院包（36 周前装箱）
├─ 宝宝出院包
└─ 安全座椅（提前装车，放婴儿用品小袋）
```

北医三院专属条目见 [北医三院产科入院清单](../待产包准备/北医三院产科入院清单.md)。

---

## 家庭操作清单

- [ ] 36 周前三大包已备好。
- [ ] 安全座椅已安装并试固定。
- [ ] 证件和产检本在一个固定位置。
- [ ] 家人知道去医院路线和产科电话。
