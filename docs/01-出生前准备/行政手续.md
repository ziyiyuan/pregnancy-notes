# 行政手续

## 这一页解决什么问题

宝宝出生后需要办理的证件、保险和保健建档。**各地时限和流程不同，以下为中国常见框架；具体以分娩医院和居住地社区要求为准。**

---

## 出生后在医院完成（通常）

| 事项 | 说明 |
|------|------|
| 出生医学证明 | 填写父母信息；名字可事先想好 |
| 新生儿听力筛查 | 医院常规 |
| 新生儿疾病筛查 | 足跟采血（苯丙酮尿症、甲低等） |
| 乙肝疫苗第 1 剂 + 卡介苗 | 按身体状况，可推迟（早产、低体重等） |
| 出院小结 | 保存好，儿保和报销要用 |

---

## 出院后 1 个月内（常见）

| 事项 | 去哪里 | 备注 |
|------|--------|------|
| 户口登记 | 派出所 | 需出生证、父母身份证、户口本等 |
| 医保参保 | 医保局 / 线上 | 新生儿随父或随母，各地政策不同 |
| 儿保建档 | 社区卫生服务中心 | 首次访视或体检预约 |
| 预防接种证 | 社区接种门诊 | 记录后续疫苗 |

---

## 生育相关（家长）

- 生育津贴 / 产假：向单位人事和当地医保部门咨询。
- 商业保险：如有孕产险、新生儿险，按保单时限报案。

---

## 建议准备的材料包

```
出生医学证明（原件 + 复印件）
出院小结
母子健康手册
父母身份证、户口本、结婚证（按当地要求）
预防接种证（社区发放或已有）
```

---

## 1 月龄前后别忘了

- [ ] 1 月龄儿保体检
- [ ] **乙肝疫苗第 2 剂**（社区接种）
- [ ] 记录体重、身长、头围

详见 [儿保与疫苗](../02-新生儿期-0-4周/儿保与疫苗.md)。

---

## 办理顺序建议

```text
出院 → 出生证 → 户口（如需）→ 医保 → 社区儿保建档 → 1 月龄体检 + 疫苗
```

---

## 出院前问清楚

- [ ] 黄疸复查时间和地点？
- [ ] 社区儿保首次访视谁联系、怎么预约？
- [ ] 卡介苗 / 乙肝是否已在院接种？若未种，何时去社区补？
- [ ] 母乳喂养门诊或哺乳顾问联系方式？
