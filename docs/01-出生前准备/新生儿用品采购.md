# 新生儿用品采购

## 使用场景

宝宝回家前完成睡眠、喂养、换洗、出行、健康护理的基础配置。**母乳为主**，配方奶仅作应急备用。

## 可选

| 物品 | 适合什么家庭 | 不买是否影响护理 |
|------|-------------|-----------------|
| 消毒锅 / 温奶器 | 配方奶或瓶喂较多 | 否，可沸水消毒 |
| 安抚奶嘴 | 需观察含乳建立后再考虑 | 否 |
| 背带 / 腰凳 | 出门多、长时间抱 | 否，0–4 周少用 |
| 床铃 / 转转乐 | 需挂牢、够高；5 月或能坐后移除 | 否 |
| 摇椅 | | 否 |

---

## 不建议

| 物品 | 不建议原因 |
|------|-----------|
| 婴儿枕头 | 窒息风险，1 岁前不需要 |
| 床围 / 防撞条 | 窒息、夹伤风险 |
| 厚重松散被子（睡觉） | 用睡袋替代 |
| 新生儿鞋 | 会走之前不需要 |
| 学步车 | 安全隐患，不促进发育 |
| 爽身粉 | 吸入风险，通常不需要 |

---

## 0–4 周最小可用集

若预算或时间有限，以下 **7 项** 即可应对出院：

1. 安全睡眠空间（床 + 床垫 + 睡袋）
2. 安全座椅
3. 纸尿裤 + 护臀膏 + 棉柔巾
4. 52 码连体衣 + 包被
5. 哺乳用品（文胸、垫、乳头霜）
6. 体温计
7. 小夜灯
