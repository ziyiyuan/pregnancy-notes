# 新生儿必备知识

## 这一页解决什么问题

在宝宝出生前，家长最需要提前建立的认知框架——知道什么是正常、什么是危险、以及和中国常见月子说法如何区分。

---

## 1. 安全睡眠（ABC）

| 原则 | 含义 |
|------|------|
| **A**lone | 单独睡眠空间（同房不同床可接受） |
| **B**ack | 仰卧 |
| **C**rib | 平坦、坚实的床面 |

✅ 睡袋优于松散毯子。  
❌ 不放枕头、床围、毛绒玩具、厚被。  
✅ 清醒时在成人看护下 **俯卧练习（Tummy Time）**。

---

## 2. 母乳喂养基础

- **产后尽早**肌肤接触；**黄金 1 小时**内尝试第一次哺乳。
- 剖宫产 / 全麻：婴儿可能嗜睡，需在清醒窗内更主动尝试含乳。
- **按需喂养**，通常 8–12 次 / 24 h，不必死盯钟点。
- 判断够不够：含乳 + 吞咽 + 湿尿布（第 4 天起 **≥6 次/日**）+ 体重回升。
- 配方奶：仅作医学指征或 **短期应急**，不随意加量。

---

## 3. 出生后正常现象

| 现象 | 说明 |
|------|------|
| 生理性体重下降 | 出生后 3–5 天，≤7–10%；约 2 周恢复出生体重 |
| 胎便 → 黄便 | 黑绿黏稠 → 过渡 → 黄色稀软（母乳） |
| 新生儿黄疸 | 常见，需观察加重速度；明显加重应就医 |
| 脱皮 | 常见，保湿即可 |
| 惊跳反射 | 正常，随月龄减弱 |
| 打嗝、少量吐奶 | 常见 |
| 不一定会大哭 | 很多新生儿安静开始呼吸 |

---

## 4. 需要警惕（出生前就要知道）

- 发热（≥38℃ 腋温，尤其 <3 月龄）
- 持续拒奶、吃奶无力
- 明显嗜睡、难唤醒
- 呼吸急促、口唇发紫
- 尿量明显减少
- 大便带血、大量黏液或水样腹泻伴精神差
- 脐部红肿、流脓、臭气

详见 [就医警示清单](../10-附录与工具/就医警示清单.md) 和 [红旗信号](../02-新生儿期-0-4周/红旗信号.md)。

## 5. 循证 vs 常见月子说法

| 常见说法 | 循证建议 |
|----------|----------|
| 不能开窗、要捂汗 | 室温舒适（约 22–26℃），适当通风，**摸颈背**判断冷热 |
| 新生儿要多喂水 | 母乳或配方奶已含水，**通常不需额外喂水** |
| 要剃胎毛 | 无医学必要 |
| 要绑腿 / 把尿 | 不建议；襁褓需 **hip-friendly**，臀部可活动 |
| 必须穿袜穿鞋 | 会走前 **不需要鞋**；袜子勿过紧 |
| 背景电视没关系 | **2 岁内尽量避免**屏幕，含背景电视 |

与长辈沟通时，可以「原则一致、方法温和」：强调宝宝安全和妈妈休息，而非对错争论。

---

## 6. 婴儿房布置要点

- 干净、无尘；可洗物品反复漂洗，头几个月 **单独洗涤**。
- 远离空调直吹、窗缝强风、取暖器。
- 留 **小夜灯**，便于夜间观察。
- 热水出水 **≤ 49℃**。
