## 3. 单篇文章写作标准

每篇护理文档不再只写泛泛原则，统一写成以下结构：

```text
# 标题

## 这一页解决什么问题

## 结论先行

## 为什么宝宝会这样

## 家长具体怎么做

## 什么是正常

## 什么时候需要观察

## 什么时候需要就医

## 家庭操作清单

## 本文依据
```

其中「家长具体怎么做」必须写到可执行，例如：

- 准备什么
- 先做什么
- 后做什么
- 多久做一次
- 观察什么
- 出现什么情况停止并就医

## 4. 0–4 周必须纳入的读书笔记要点

| 笔记主题 | 写入位置 |
|---|---|
| 产后 1 小时内开始哺乳、剖宫产全麻可能影响新生儿清醒度 | `吃.md`、`开奶.md` |
| 出生后第一个月尽量回应哭闹，不刻意让新生儿哭 | `睡.md`、`陪.md` |
| 清醒 1–2 小时后就可能需要再次入睡 | `睡.md` |
| 仰卧睡眠适用于 1 岁以内，半岁内尤其重要 | `睡.md`、`安全睡眠.md` |
| 清醒时看护下俯卧练习 | `睡.md`、`陪.md` |
| 胎便、过渡便、母乳便、配方奶便 | `拉.md` |
| 粉色尿渍、尿量、尿色、血尿 | `撒.md` |
| 脐带未脱落前擦浴、每周约 3 次、热水≤49℃ | `洗.md` |
| 婴儿衣物比成人多一层，不穿鞋，袜子不要过紧 | `穿.md` |
| 2 岁前尽量不看屏幕，包括背景电视 | `陪.md` |
| 婴儿床不要放软物，床铃 5 月龄或会坐后移除 | `新生儿用品采购.md`、后续安全专题 |
