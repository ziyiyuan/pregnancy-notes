# 蔚然成长记录：0–6岁科学育儿指南 - 实现方案 v1.0

## 1. 项目定位

本项目定位为一个面向中国家庭、可长期维护、可发布到 GitHub Pages 的科学育儿知识库。

当前阶段重点完成：

> 0–1岁科学育儿指南：中国家庭版

长期目标扩展至：

- 待产准备
- 新生儿期
- 0–1岁婴儿护理
- 1–3岁幼儿养育
- 3–6岁儿童成长
- 家庭安全与急救
- 常见问题 FAQ
- 蔚然个人成长记录

## 2. 技术方案

采用现有 `pg` 项目作为文档站点基础：

- 文档框架：MkDocs
- 主题：Material for MkDocs
- 源目录：`docs/`
- 配置文件：`mkdocs.yml`
- 发布方式：GitHub Pages
- 自动部署：GitHub Actions

继续使用 MkDocs，而不是重新建立 GitBook 项目。理由：

1. 当前项目已经具备 MkDocs 发布基础。
2. Material for MkDocs 对中文、搜索、导航和长期维护更友好。
3. 适合构建类似技术文档的知识库结构。
4. 后续可持续扩展到 3 岁、6 岁甚至更长期。

## 3. 内容原则

全书统一采用以下结构：

```text
发育 → 护理 → 常见问题 → 危险信号 → 工具与记录
```

每个年龄段重点回答：

1. 宝宝这个阶段身体和生理发生了什么变化？
2. 这些变化对应的行为表现是什么？
3. 家长应该如何护理？
4. 什么情况是正常的？
5. 什么情况需要观察？
6. 什么情况需要就医？
7. 家里需要准备哪些东西？
8. 如何记录喂养、睡眠、排便、疫苗和发育？

## 4. 证据等级

| 标识 | 含义 | 示例来源 |
|---|---|---|
| ✅ 强证据 | 国际或国家级权威建议 | AAP、WHO、CDC、中国国家免疫规划 |
| 🟡 中等证据 | 专家共识、临床指南 | 儿科共识、临床指南 |
| ⚪ 经验建议 | 家庭护理经验、操作建议 | 家庭分工、护理技巧 |

医学和护理建议应尽量注明依据。不得大段复制受版权保护的图书原文，应以归纳、转述、整理为主。

## 7. 长期维护方式

### 通用知识

放入各阶段和专题目录，例如：

- `02-新生儿期-0-4周/睡.md`
- `08-家庭安全与急救/安全睡眠.md`
- `09-FAQ知识库/睡眠FAQ.md`

### 个人记录

放入：

- `90-成长记录/`

用于记录蔚然自己的出生、体重、身高、疫苗、第一次翻身、第一次爬行等。

### 写作模板

放入：

- `99-写作模板/`

用于后续 AI 辅助写作，保证风格、结构和证据等级一致。
